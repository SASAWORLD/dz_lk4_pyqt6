# Проект GUI приложений на PyQt6

Этот проект содержит два графических приложения, созданных с использованием фреймворка PyQt6 для Python.

## Стек технологий

### Основные технологии:
- **Python 3.x** - основной язык программирования
- **PyQt6** - фреймворк для создания графических интерфейсов
- **Qt6** - кроссплатформенная библиотека для разработки GUI

### Ключевые компоненты PyQt6:
- `QApplication` - главный класс приложения
- `QWidget` - базовый класс для всех объектов интерфейса
- `QLabel` - виджет для отображения текста и изображений
- `QPixmap` - класс для работы с растровой графикой
- `QPainter` - инструмент для рисования и преобразования изображений
- `QFont` - работа со шрифтами

### Особенности реализации:

#### В файле `planet.py`:
- Базовое использование QLabel для текста и изображений
- Обработка исключений при загрузке файлов
- Простое позиционирование элементов

#### В файле `profile.py`:
- Продвинутая работа с изображениями:
  - Обрезка и масштабирование изображений
  - Создание круглых аватарок с антиалиасингом
  - Стилизация CSS-like для фоновых элементов
- Использование различных шрифтов и размеров текста
- Многострочный текст с переносом слов
- Структурированная компоновка интерфейса

## Требования

- Python 3.6+
- PyQt6
- Доступ к графической среде (для отображения GUI)
